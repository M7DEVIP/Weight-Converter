## 🚀 Quick Start

Get up and running in less than 30 seconds!

### Method 1: Direct Browser Opening (Easiest)
1. **Download/Clone** this repository
2. **Open** `index.html` in your web browser
3. **Start converting!** 🎉

### Method 2: Live Server (Recommended)
1. **Clone** the repository
2. **Open** with VS Code
3. **Install** Live Server extension
4. **Right-click** on `index.html` → **Open with Live Server**
5. **Enjoy** with hot-reload! 🔥

## 💻 Installation

### Prerequisites
- ✅ **Web browser** (Chrome, Firefox, Safari, Edge)
- ✅ **Optional:** VS Code with Live Server extension

### Step-by-Step Installation

#### **Option A: Browser Method** 🖥️
```bash
# 1. Clone the repository
git clone https://github.com/M7DEVIP/Weight-Converter.git

# 2. Navigate to the project
cd Weight-Converter

# 3. Open in browser
open index.html  # macOS
# OR simply double-click index.html
```

#### **Option B: Live Server** ⚡
```bash
# 1. Install VS Code
# 2. Install Live Server extension from marketplace
# 3. Open project in VS Code
# 4. Right-click index.html → "Open with Live Server"
```

#### **Option C: Python Server** 🐍
```bash
# 1. Navigate to project directory
cd Weight-Converter

# 2. Start Python server
python -m http.server 8000

# 3. Open http://localhost:8000 in your browser
```

### Verification ✅
Once installed, you should see:
- Weight Converter title
- Four input fields (kg, g, lbs, mg)
- Theme toggle switch
- Convert and Clear buttons

## 📖 Usage

### Basic Conversion
1. **Enter** a value in any weight field
2. **Watch** real-time conversion in other fields
3. **Toggle** between light/dark themes using the switch
4. **Clear** all fields with the Clear button

### Example
```
Input: 1 kg
Output: 1000 g | 2.20462 lbs | 1000000 mg
```

### Tips 💡
- ✅ Values update automatically as you type
- ✅ Data persists between browser sessions
- ✅ Works offline once loaded
- ✅ No internet connection required
